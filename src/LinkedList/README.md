# Linked List

특징

- 특정 위치에서 요소를 삽입하거나 삭제하는 작업이 배열보다 효율적입니다.
  배열에서는 중간에 요소를 삽입하거나 삭제할 때 나머지 요소를 이동시켜야 하지만, Linked List는 `포인터만 변경`하면 됩니다.
